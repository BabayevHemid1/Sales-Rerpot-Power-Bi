## Overview
This repository contains a dynamic Power BI project (`Sales Report.pbix`) developed to analyze and visualize enterprise sales data. The dashboard provides an intuitive interface for stakeholders to track Key Performance Indicators (KPIs), monitor sales trends, and discover actionable insights for business growth.

## Key Features
* **Executive Summary:** A high-level view of Total Sales, Profit, and Order Quantities.
* **Trend Analysis:** Detailed time-series visuals tracking performance across different quarters and months.
* **Geographical Insights:** Regional breakdown of sales to identify top-performing markets.
* **Product & Category Breakdown:** Analysis of the most and least profitable product lines.
* **Interactive Slicers:** Dynamic filtering capabilities allowing users to drill down by date, location, and product category.

## Tools & Technologies Used
* **Power BI Desktop:** Core tool for data modeling and dashboard creation.
* **Power Query:** Utilized for ETL (Extract, Transform, Load) processes, ensuring data quality and consistency.
* **DAX (Data Analysis Expressions):** Implemented for complex aggregations, time-intelligence calculations, and custom metrics.

## How to Use This Project
1. Ensure you have [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed on your machine.
2. Clone this repository or download the `.pbix` file directly.
3. Open `Sales Report.pbix` in Power BI Desktop.
4. Use the filter panels and interactive charts to explore the data.
